CentOS 安装 Git

CentOS 使用 yum 安装 git 默认版本是 v1.8.3.1

安装 git v2.x

首先卸载当前git

yum remove git

安装指定版本 git

执行 make prefix=/usr/local/git all 可能会报

此时需要安装 openssl

yum install openssl*

再执行

执行 make prefix=/usr/local/git install 报错
